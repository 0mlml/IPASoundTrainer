# Ipa Sound Trainer

## Idea
The player learns how to recognise similar IPA sounds 
(like in words first/thirst) in a fun interactive way.

The idea comes from my previous work done in jupyter notebook [pronunciation.ipynb](pronunciation.ipynb),
which I created to practice sound recognition. Frankly, I'm still bad at it.

## Instructions
Audio should be on.

The player is represented by a moon icon. 
They can move using arrow keys. 

After overlapping with a gift object an audio is played.
The player has some time to move to the correct answer area.

When the time elapses, a player will:
- gain a point if they are in the correct answer area
- lose a heart if they are in the wrong answer area
- get nothing if they are in both areas

After all hearts are lost, the game displays a game over
screen and then restarts itself.
The game over screen lasts for three seconds showing the final score.
